CSS stands for Cascading Style Sheets is a style sheet language used for describing the presentation of a document written in a markup language like HTML. CSS is a cornerstone technology of the World Wide Web. It is a simple design language intended to simplify the process of making web pages presentable.

CSS defines the style of HTML. for eg.
  • make all text purple
  • give the second image a yellow border
CSS is basically the adjectives of a webpage or skin to HTML skeleton .

There are three ways to write css styles either Inline or using the style tag or separating css and html.

eg. of inline is :

eg. of style tag is :
style type="text/css"
selector

CSS Selectors

CSS selectors are used to "find" (or select) HTML elements based on their element name, id, class, attribute and more.

The element Selector

The element selector selects elements based on the element name. You can select all elements on a page or some specific elements . eg.

h1 {
background: pink;
border: 2px solid black;
}
This will set the background of heading 1 to be pink and a black border of 2px black color .

The id Selector

The id selector uses the id attribute of an HTML element to select a specific element. The id of an element should be unique within a page, so the id selector is used to select one unique element! To select an element with a specific id, write a hash (#) character, followed by the id of the element.
The id for the paragraph is considered as "special" eg.

#special {
font-family: Arial;
font-size: 20px;
background: #FFA07A;
}

The class Selector

The class selector selects elements with a specific class attribute. To select elements with a specific class, write a period (.) character, followed by the name of the class. In the example below, all HTML elements with class="center" will be red and center-aligned:

.center {
color: red;
text-align: center;
}

Star Selector

The star selector selects everything in the webpage and gives a general style to it. eg.
* {
font-family: Calibri;
color : blue;
}
This will set the whole text of the page to be blue and of the above style .

Rational Selectors

Rational Selectors targets elements based on their relationship with another elements. eg .

Descendant Selector : eg.

div p {
color : blue;
}
This targets all p tags that are descendent (child, grandchild, great-grandchild, and so on) of div tag.

Adjacent Sibling Selector : eg.

h4 + p {
text-align: right;
}
This selector matches p tag next to h4 and both sharing same parent. This means p should come directly after h4.

General Sibling Selector : eg.

h2~p {
border: 2px solid blue;
}
This selector matches all p elements exactly after h2 element.

Attribute Selectors

They targets matching elements based on their attributes and attributes value. Attribute Selectors are used in Brackets [].
eg .



a[href="style.css"] {
background: yellow;
}

This will give a yellow background to the anchor tag .
div {
class^="col"
}

This selector matches only div element with class value starting exactly with "col" .

div{
class$="wrap"
}
This selector matches only element div with class ending exactly with "wrap" .

div{
class*="md"
}

This selector matches only element div with class that contains the substring "md" .

Pseudo Selectors

CSS Pseudo Selectors are classified in Pseudo Class and pseudo elements. CSS Pseudo Selectors are called using Colon : .Pseudo elements are called using double colon ::

There are currently seven pseudo-elements in CSS. They are:
::after
::before
::first-letter
::first-line
::marker
::placeholder
::selection

eg .

h3:hover
{
color: blue;
}
h3 will change to a different color when hovered over .

a:visited
{
color: gray;
}
When the anchor tag links are visited they will turn grey .

#special::after {
content: “ — 1”;
color: blue;
}
The string is added after the paragraph with id special and their colors are updated .


The attributes that we use inside selectors are :


color : It sets the color of the text in the element . eg.

color: blue;

background : It sets the background color for the particular element . eg.

background: pink;

The background can also be added using a url like .
background: url(link address); eg .

background: url(data:image/jpeg;base64,/9j/4AAQSkZJRgABAQAAAQABAAD/2wCEAAkGBwgHB);

border : It forms a border around the element with particular width and color and design . eg .

border: 2px dashed black;

padding : It sets the vertical distance between the border and the text within . eg .

padding: 20px;

font-family : It helps us to choose which font style do we want to use for our text .Font styles can also be added in CSS using Google Fonts .

eg.
font-family: Verdana;

font-size: It specifies the font size . eg.

font-size: 20px;

text-align: It describes the alignment of the text in the paragraph . eg .

text-align: right;

line-height: It specifies the distance between two lines in a paragraph .eg.

line-height: 1.2px;

margin : It sets the vertical distance between the border and the text within and the overall alignment of the border . There are more attributes like margin-left , margin-bottom , margin-right , margin-top .

eg.
margin: 20px auto;

text-transform: uppercase : It transforms all the text into uppercase.

text-transform: lowercase : It transforms all the text into lowercase.

text-transform: capitalize : It transforms the first letter of the text into uppercase.

text-decoration: The text-decoration property specifies the decoration added to text .It can be overline, line-through, underline, underline overline or any other such attribute .

Similar to the selector, there are lot many attributes that help us to style pages .
