HTML is an acronym for Hypertext Markup Language which is used for displaying pages or documents in a web browser like Google Chrome , Mozilla Firefox , Internet Explorer , etc.
HTML defines the structure of a webpage . for eg .
  • put an image here
  • put a form / paragraph here
HTML is basically the nouns / skeleton of a webpage .
The latest version of HTML is HTML5.

HTML elements are mostly constituted by tags, written using angle brackets < >. Tags are a set of formatted commands for a web page .Most of the tags in HTML are paired tags,i.e. these tags have to be closed after using them.Some tags donot need to be closed after using them which are unpaired tags.The additional properties of a tag are defined by attributes of that tag . The list of tags in HTML are :

1. html tag : All web pages start and end with the html tag. All the other tags come inside this tag.Hence it is also called as the root element.

2. head tag : It is used to contain all the head elements in the html file. It contains the title, style, meta, etc tag.

3. title tag : It is used to define the title of the html document .

4. body tag : It is used to define the body of html document.It contains image, tables, lists, ...etc.

5. paragraph tag : It is used to define paragraph content in html document.

6. font tag : It is used to specify the font size, font color and font-family in html document .

7. center tag : It is used to set the content into the center .

8.emphasis tag : It is used to emphasize or put more significance on a text. eg. This text is emphasized

9. bold tag : It is used to specify bold content in html document .

10. italic tag : It is used to write the content in italic format.

11. subscript tag : It is used for adding a subscript text in the html file.Like H20 can be written as H2O

12. superscript tag : It is used for adding a superscript text in the html file.Like x2 can be written as x2

13. underline tag : It is used to set the content underline.
eg. This text is underlined
